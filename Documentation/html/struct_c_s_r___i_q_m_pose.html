Detailed Description

Inter-Quake Model pose

Note
Channels 0..2 are translation <Tx, Ty, Tz> and channels 3..6 are quaternion rotation <Qx, Qy, Qz, Qw>. Rotation is in relative/parent local space. Channels 7..9 are scale <Sx, Sy, Sz>, so: output = (input * scale) * rotation + translation
