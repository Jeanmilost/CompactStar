26// IMPORTANT NOTE
27// This Collada reader is partially implemented and planned to read simple files
28// exported from Blender. For that reason there are several known limitations,
29// which are (among others):
30// - Only one skeleton is supported per scene
31// - In geometries, only one set of triangles is supported per mesh
32// - Only triangles are supported in meshes, i.e no <trifans>, <tristrips>, ...
33// - Triangles always begin with VERTEX as input, and coords are always [x, y, z]
34// - Only textures are read in materials
35// ...
36// If one of the above condition is not respected, the Collada model may fail to
37// be read
