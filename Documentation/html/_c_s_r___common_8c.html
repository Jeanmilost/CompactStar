◆ csrMemoryAlloc()

void * csrMemoryAlloc ( void *  pMemory,
size_t  size,
size_t  count 
)

Allocates or reallocates a new block of memory

Parameters
pMemory- previous memory block to reallocate, if 0 a new block will be allocated
size- size of a single item in the memory block, in bytes
count- number of items the memory block will contain
Returns
newly allocated or reallocated memory block, 0 on error
Note
If the newly allocated size is smaller than the existing one, the remaining extra memory will be freed internally
The memory pointed by pMemory is preserved in case the reallocation failed
The new memory block should be freed with the free() function when becoming useless
