typedef unsigned(* CSR_fOnGetVertexColor) (const CSR_VertexBuffer *pVB, const CSR_Vector3 *pNormal, size_t groupIndex)

Called when a vertex color should be get

Parameters
pVB- vertex buffer that will contain the vertex for which the color should be get
pNormal- vertex normal
groupIndex- the vertex group index (e.g. the inner and outer vertices of a ring)
Returns
RGBA color to apply to the vertex
Note
This callback will be called only if the per-vertex color option is activated in the vertex buffer
